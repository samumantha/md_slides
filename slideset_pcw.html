Point clouds

set of data points in 3D space

Sources:

  • spaceborne / airborne / terrestrial / mobile LiDAR
  • photogrammetry (from images: structure from motion / multi view stereo ...)

File formats:

  • ASCII: XYZ, ASC, PTX, ...
  • binary: LAS/LAZ , FLS, PCD, ...
  • other: PLY, FBX, E57 , OBJ, ...
